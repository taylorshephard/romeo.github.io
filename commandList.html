Command Cheat Sheet

Republish lodown project to npm

Update the version number in your lodown package.json file

in the terminal, (~/environment/lodown) run

npm publish

in another terminal (~/enviornment/gitHubName.github.io) run

npm install underscore

npm link underscore

Change into your repository folder

cd "your-github-username"-github.io


Pushing changes to GitHub

git add -A (To add all the files you want to add to github)

git status (To see the files you are about to add to github)

git commit -m "<\your commit message>" (To attach a message to all the files)

git push (To push all of your work up to github)


Change out of a directory

cd ..


Move files or folders in Cloud9

mv <\current-location> <\desired-location>
